## ❓ Frequently Asked Questions

### 🔧 Connection Issues

**Q: Unable to connect to server?**

A: Please check:

- Subscription link is correct and valid
- Selected node is available
- Local network connection is normal
- VPN permission has been granted

**Q: Connection speed is slow?**

A: Optimization suggestions:

- Try switching to other server nodes
- Select servers with lower latency
- Check local network quality
- Avoid peak network usage times

### 📱 Application Issues

**Q: Application crashes frequently?**

A: Solutions:

- Restart device
- Reinstall application
- Check iOS version compatibility
- Clear device storage space

**Q: Subscription import failed?**

A: Troubleshooting steps:

- Confirm subscription link format is correct
- Check network connection status
- Try manually adding single node
- Contact service provider for confirmation
