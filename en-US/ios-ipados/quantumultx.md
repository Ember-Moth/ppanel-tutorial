## ❓ Frequently Asked Questions

### 🔧 Configuration Issues

**Q: Subscription import failed?**

A: Troubleshooting steps:

- Verify subscription link format is correct
- Check network connectivity
- Try manual node addition
- Contact service provider for verification

**Q: Scripts not working?**

A: Common solutions:

- Enable JavaScript engine in settings
- Check script syntax errors
- Verify script compatibility with current version
- Consult script documentation

### 📱 Performance Issues

**Q: High battery consumption?**

A: Optimization tips:

- Disable unnecessary scripts
- Reduce update frequency
- Use appropriate policy groups
- Close unused advanced features

**Q: Slow connection speeds?**

A: Performance optimization:

- Select geographically closer nodes
- Use load balancing policies
- Optimize rule configurations
- Check local network quality
