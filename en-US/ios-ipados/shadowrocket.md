## 🚀 Configuration Tutorial

### 📋 Prerequisites

Before starting configuration, please ensure:

- ✅ Shadowrocket is successfully installed
- ✅ You have valid server information
- ✅ Network connection is normal

### 🛠️ Import Configuration

#### 🔗 Method 1: Subscription Link Import

##### Step 1: Open Application

Launch Shadowrocket and enter the main interface:

![Application Main Interface](shadowrocket-01.png)

##### Step 2: Add Subscription

Click the `+` icon in the top right corner, select `Subscribe`:

![Add Subscription](shadowrocket-02.png)

##### Step 3: Paste Link

Paste the subscription link in the URL box, click Done:

![Paste Subscription Link](shadowrocket-03.png)

##### Step 4: Update Subscription

Swipe the subscription entry, click `Update` to refresh nodes:

![Update Subscription](shadowrocket-04.png)

#### 📱 Method 2: QR Code Scanning

1. Click the scan icon in the top left corner
2. Scan the QR code provided by the service provider
3. Automatically recognize and import configuration

#### ⌨️ Method 3: Manual Addition

1. Click `+` and select `Add Server`
2. Choose the corresponding protocol type
3. Fill in detailed server information

## 🌐 Usage Guide

### 🎯 Connect to Server

1. **Select Node**: Choose appropriate server in main interface
2. **Start Connection**: Click the top switch to enable proxy
3. **Confirm Permissions**: First use requires VPN permission grant
4. **Verify Connection**: Check VPN icon in status bar

### ⚙️ Basic Settings

#### 🔧 Global Routing

- **Configuration**: Configuration → Default
- **Proxy**: All traffic through proxy
- **Direct**: All traffic direct connection
- **Scene**: Smart splitting (recommended)

#### 📊 Rule Configuration

- **Domain Rules**: Traffic splitting by domain name
- **IP Rules**: Traffic splitting by IP address
- **GeoIP**: Traffic splitting by geographical location

## ❓ Frequently Asked Questions

### 🔧 Connection Issues

**Q: What to do if connection fails?**

A: Troubleshooting steps:

- Confirm server information is correct
- Check network connection status
- Try switching to other nodes
- Restart application or device

**Q: Slow speed or unstable connection?**

A: Optimization suggestions:

- Select nodes with lower latency
- Avoid peak network usage times
- Check local network quality
- Contact service provider customer service

### 📱 Application Issues

**Q: App crashes or freezes?**

A: Solutions:

- Restart application
- Restart device
- Check iOS version compatibility
- Clear device storage space

**Q: Cannot download application?**

A: Notes:

- Confirm using foreign region Apple ID
- Check if account is valid
- Try switching to different region account
