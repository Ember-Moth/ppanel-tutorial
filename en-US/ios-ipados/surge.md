## 🎯 Professional Configuration

### 📋 Advanced Rules

Surge supports the most complex rule configurations:

```ini
# Process-based rules
PROCESS-NAME,WeChat,DIRECT
PROCESS-NAME,QQ,DIRECT

# User-agent based rules
USER-AGENT,*WeChat*,DIRECT

# Time-based rules
TIME,9:00-18:00,WORK-PROXY

# Network environment based rules
NETWORK,WIFI:HOME,DIRECT
NETWORK,CELLULAR,PROXY
```

### 🎭 Script Features

JavaScript script extension support:

```javascript
// HTTP request modification script
let headers = $request.headers;
headers['User-Agent'] = 'Custom-Agent';
$done({headers});

// HTTP response modification script
let body = $response.body;
let obj = JSON.parse(body);
obj.vip = true;
$done({body: JSON.stringify(obj)});
```

## ❓ Frequently Asked Questions

### 🔧 Technical Issues

**Q: Unable to install MitM certificate?**

A: Solution steps:

- Generate new CA certificate
- Complete certificate installation following the wizard
- Trust certificate in system settings
- Confirm certificate is enabled in "Certificate Trust Settings"

**Q: Rules not working?**

A: Check points:

- Confirm rule syntax is correct
- Check rule priority order
- Verify matching conditions are accurate
- Check real-time logs to confirm matching

### 💰 Purchase Issues

**Q: Is it worth purchasing?**

A: Evaluation points:

- Are you a professional user or developer
- Do you need detailed network analysis features
- Is budget sufficient ($49.99)
- Do you have time to learn complex features
