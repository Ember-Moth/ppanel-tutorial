# ⚔️ Clash Meta for Android - 强大的开源代理工具

> 🌟 [Clash Meta for Android](https://github.com/MetaCubeX/ClashMetaForAndroid) 是基于 Clash Meta 核心开发的 Android 代理客户端，提供了强大的规则引擎和丰富的功能特性。

## ✨ 产品特色

### 🎯 核心优势

- 🆓 **完全免费**：开源项目，永久免费使用
- 🚀 **性能卓越**：基于 Clash Meta 内核，性能优异
- 🎨 **界面现代**：Material Design 设计规范
- 🔧 **功能丰富**：支持多种高级功能和配置
- 🛡️ **安全可靠**：开源透明，社区维护

### 🔗 协议支持

| 协议 | 支持状态 | 特性说明 |
|------|----------|----------|
| 🔒 Shadowsocks (SS) | ✅ | 经典协议，稳定可靠 |
| 🔓 ShadowsocksR (SSR) | ✅ | 增强功能，混淆支持 |
| 🧦 SOCKS5 | ✅ | 通用代理协议 |
| 🐚 Snell | ✅ | 高性能协议 |
| 🌟 V2Ray (VMess/VLESS) | ✅ | 现代化协议套件 |
| 🛡️ Trojan | ✅ | TLS 加密伪装 |
| 🚀 Hysteria | ✅ | 基于 QUIC 的高速协议 |
| 🔐 WireGuard | ✅ | 现代 VPN 协议 |

### 📱 系统要求

- **最低版本**：Android 7.0 (API 24) 及以上
- **推荐版本**：Android 9.0 及以上
- **适用设备**：安卓手机 / 平板电脑
- **架构支持**：ARM64、ARM、x86_64
- **存储需求**：约 100MB 可用空间

## 🚀 使用教程

### 📋 配置步骤

#### 🔥 步骤总览

1. **📱 启动应用** - 打开 Clash Meta for Android
2. **⚙️ 导入配置** - 添加订阅链接或配置文件
3. **🌐 选择节点** - 从节点列表中选择服务器
4. **🚀 开启代理** - 启动代理服务
5. **✅ 验证连接** - 确认代理功能正常

### 🎯 详细操作

#### 🌟 步骤一：应用启动

打开已安装的 Clash Meta，进入主界面：

点击"配置"标签，进入配置管理页面：

![配置管理页面](clash-meta-for-Android-02.jpg)

#### 📥 步骤三：导入配置

点击右上角的"+"号，选择导入方式：

![导入配置选项](clash-meta-for-Android-03.jpg)

#### 📝 步骤四：配置信息

输入配置信息（订阅链接或配置文件）：

![配置信息填写](clash-meta-for-Android-04.jpg)

#### ⏳ 步骤五：等待加载

配置导入后，等待节点信息加载完成：

![配置加载过程](clash-meta-for-Android-05.jpg)

#### 🌐 步骤六：选择节点

在节点列表中选择合适的服务器：

#### ✅ 步骤七：确认配置

检查配置信息，确认无误后保存：

![配置确认界面](clash-meta-for-Android-07.jpg)

#### 🚀 步骤八：启动服务

返回主界面，开启代理服务：

## 🎛️ 高级功能

### 📊 规则管理

- **域名规则**：基于域名的智能分流
- **IP 规则**：基于 IP 地址的精准分流
- **GeoIP 规则**：基于地理位置的自动分流
- **自定义规则**：支持用户自定义分流规则

### 🔄 策略组配置

- **自动选择**：根据延迟自动选择最优节点
- **负载均衡**：流量在多个节点间均衡分配
- **故障转移**：节点失效时自动切换备用节点
- **手动选择**：用户手动指定使用的节点

### 📈 监控和统计

- **实时流量**：显示当前上传/下载速度
- **连接信息**：查看活跃连接详情
- **日志记录**：详细的运行日志信息
- **规则匹配**：显示规则命中情况

## ❓ 常见问题

### 🔧 技术支持

**Q: 无法导入配置？**

A: 请检查：

- 订阅链接是否正确
- 网络连接是否正常
- 配置格式是否兼容

**Q: 连接后无法上网？**

A: 排查建议：

- 检查节点是否可用
- 确认 VPN 权限已授予
- 尝试切换不同节点

**Q: 应用崩溃或卡顿？**

A: 解决方案：

- 重启应用或设备
- 清除应用缓存数据
- 检查系统资源占用
