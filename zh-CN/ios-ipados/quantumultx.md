## 🎛️ 实用配置

### 📋 分流规则

智能分流配置示例：

```ini
# 国内网站直连
HOST-SUFFIX,baidu.com,DIRECT
HOST-SUFFIX,qq.com,DIRECT
GEOIP,CN,DIRECT

# 国外网站代理
HOST-SUFFIX,google.com,PROXY
HOST-SUFFIX,youtube.com,PROXY
GEOIP,US,PROXY

# 最终规则
FINAL,PROXY
```

### 🔄 策略组配置

```ini
# 自动选择最优节点
url-latency-benchmark=Auto,服务器1,服务器2,服务器3,url=http://www.gstatic.com/generate_204,interval=600

# 手动选择
static=Manual,服务器1,服务器2,服务器3

# 故障转移
available=Fallback,服务器1,服务器2,服务器3,url=http://www.gstatic.com/generate_204,interval=600
```

## ❓ 常见问题

### 🔧 配置问题

**Q: 脚本无法正常运行？**

A: 检查要点：

- 确认脚本语法正确
- 检查脚本权限设置
- 验证触发条件是否满足
- 查看脚本运行日志

**Q: MitM 证书安装失败？**

A: 解决步骤：

- 前往 设置 → 通用 → 描述文件
- 找到 Quantumult X 证书并信任
- 前往 设置 → 通用 → 关于本机 → 证书信任设置
- 开启对 Quantumult X 证书的完全信任

### 📱 使用问题

**Q: 应用消耗电量大？**

A: 优化建议：

- 合理配置分流规则
- 减少不必要的脚本运行
- 适当调整更新频率
- 关闭不需要的高级功能

**Q: 配置太复杂，如何简化？**

A: 简化方案：

- 使用 Gallery 导入成熟配置
- 从基础功能开始，逐步学习
- 关注社区教程和经验分享
- 考虑使用更简单的替代方案
