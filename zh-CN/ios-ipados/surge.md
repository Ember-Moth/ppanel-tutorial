## 🎯 专业配置

### 📋 高级规则

Surge 支持最复杂的规则配置：

```ini
# 基于进程的规则
PROCESS-NAME,WeChat,DIRECT
PROCESS-NAME,QQ,DIRECT

# 基于用户代理的规则
USER-AGENT,*WeChat*,DIRECT

# 基于时间的规则
TIME,9:00-18:00,WORK-PROXY

# 基于网络环境的规则
NETWORK,WIFI:HOME,DIRECT
NETWORK,CELLULAR,PROXY
```

### 🎭 脚本功能

支持 JavaScript 脚本扩展：

```javascript
// HTTP 请求修改脚本
let headers = $request.headers;
headers['User-Agent'] = 'Custom-Agent';
$done({headers});

// HTTP 响应修改脚本
let body = $response.body;
let obj = JSON.parse(body);
obj.vip = true;
$done({body: JSON.stringify(obj)});
```

### 🔄 策略组

复杂的策略组配置：

```ini
# URL 测试组
url-test = Auto,Server1,Server2,Server3,url = http://www.gstatic.com/generate_204

# 负载均衡组
load-balance = Balance,Server1,Server2,Server3,persistent = true

# 故障转移组
fallback = Fallback,Server1,Server2,Server3,url = http://www.gstatic.com/generate_204
```

## ❓ 常见问题

### 🔧 技术问题

**Q: MitM 证书无法安装？**

A: 解决步骤：

- 生成新的 CA 证书
- 按照向导完成证书安装
- 在系统设置中信任证书
- 确认证书在"证书信任设置"中已启用

**Q: 规则不生效？**

A: 检查要点：

- 确认规则语法正确
- 检查规则优先级顺序
- 验证匹配条件是否准确
- 查看实时日志确认匹配情况

### 💰 购买问题

**Q: 是否值得购买？**

A: 评估要点：

- 是否为专业用户或开发者
- 是否需要详细的网络分析功能
- 预算是否充足（$49.99）
- 是否有时间学习复杂功能

**Q: 试用期限制？**

- 提供7天完整功能试用
- 试用期内可体验所有功能
