# ClashX Pro

[ClashX Pro](https://clashxhub.com/apps/ClashXPro.dmg) 是一款现代化的 macOS 代理工具，支持：

- Shadowsocks (SS)
- V2Ray (VMess)
- Trojan
- HTTP/HTTPS
- SOCKS5
- SOCKS5 over TLS
