# ⚡ Clash for Windows

> 🚀 **经典代理客户端** | 官方图形界面，跨平台支持，功能强大稳定

[Clash for Windows](https://github.com/clashdownload/Clash_for_Windows) 是基于 Clash 内核的官方 Windows 图形客户端，提供跨平台支持（Windows、Linux、macOS），集成多种主流代理协议，为用户提供稳定可靠的代理服务。
