# 🎮 Netch - 游戏加速专用工具

> 🚀 **专为游戏优化的代理工具** | 进程级精准代理，专注游戏网络加速

[Netch](https://github.com/netchx/netch) 是一款专门为 Windows 平台游戏用户设计的开源网络代理工具，通过智能扫描游戏目录获取需要代理的进程名，实现进程级别的精准网络代理。

## 🎯 产品特色

### 🎮 游戏专用设计

- **进程级代理**：仅代理游戏相关进程，系统其他应用不受影响
- **自动游戏识别**：智能扫描游戏目录，自动获取需要代理的进程
- **游戏模式优化**：专为游戏网络环境优化的连接策略
- **延迟优化**：减少游戏网络延迟，提升游戏体验

### ⚡ 性能优势

- **轻量级设计**：占用系统资源少，不影响游戏性能
- **快速启动**：启动速度快，即开即用
- **稳定连接**：连接稳定，减少游戏掉线情况
- **智能路由**：自动选择最优网络路径

## ❓ 常见问题

### 🔧 安装使用

**Q: 如何解压和运行 Netch？**

A: 使用步骤：

1. 下载 7z 压缩包到本地
2. 使用 7-Zip 或 WinRAR 解压文件
3. 进入解压目录，双击 Netch.exe 运行
4. 首次运行可能需要管理员权限

**Q: 软件启动后无响应怎么办？**

A: 可能的解决方案：

1. 以管理员身份运行
2. 检查防火墙和杀毒软件设置
3. 确保 .NET Framework 已安装
4. 重新下载解压软件包

### 🎮 游戏相关

**Q: 如何为特定游戏配置代理？**

A: 配置步骤：

1. 在模式页面创建新模式
2. 添加游戏的主程序文件
3. 选择合适的代理服务器
4. 启动模式并运行游戏

**Q: 游戏延迟没有改善怎么办？**

A: 优化建议：

1. 更换不同的代理服务器节点
2. 调整代理模式配置
3. 检查本地网络环境
4. 尝试其他游戏加速方案
